# parkinglot

# Requirements
  1. Latest version of JDK
  2. Latest version of maven
 
 # How to build
  Run the following command
  > mvn clean install
 
 # How to Run Project
  Run the following commands
  > $ java -jar bin/Apps.jar (interactive version)
  > $ java -jar bin/Apps.jar file_inputs.txt (file version)
